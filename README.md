# local-temperature-server

- 개발 100%
- 디자인 100%

## 1. 기능
- 웹서비스 URL 파라미터 생성 및 호출 (특정 지역, 현재 시간)
- 결과 파싱해서 txt 파일로 저장 (기온 데이터)
- [Client용 프로그램](https://github.com/yeaeun23/local-temperature-client)
